# Attributions données

FoodNote peut utiliser des sources de données externes. Ces sources ne sont pas incluses comme propriété du projet FoodNote.

## CIQUAL / Anses

La table CIQUAL est une source de composition nutritionnelle des aliments maintenue par l'Anses.

Attribution recommandée dans l'application ou la documentation :

```text
Données nutritionnelles : Table Ciqual — Anses.
```

## OpenFoodFacts

OpenFoodFacts fournit des données ouvertes sur les produits alimentaires, notamment les produits scannés par code-barres.

Attribution recommandée :

```text
Données produits : OpenFoodFacts.
```

## Important

Les fichiers générés ou téléchargés localement ne doivent pas forcément être commités dans GitHub. Respectez les licences propres à chaque source de données si vous redistribuez des exports ou bases enrichies.
